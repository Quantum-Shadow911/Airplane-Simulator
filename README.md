# ✈️ Airplane Simulator

A simple 3D airplane simulator built with **React**, **Vite**, **Three.js**, and **React Three Fiber**.

This project is a miniature flight simulator where you can fly over a stylized San Francisco-inspired environment featuring an ocean, skyline, and a simplified Golden Gate Bridge. It also includes basic aircraft and airline selection.

## Features

* 🌎 3D environment
* ✈️ Flying airplane animation
* 🌉 Golden Gate Bridge replica
* 🏙️ San Francisco-style skyline
* 🌊 Ocean
* 🛫 Aircraft selection
* 🎨 Airline selection
* ⚡ Built with React + Vite
* 🎮 Powered by Three.js and React Three Fiber

## Technologies

* React
* Vite
* Three.js
* React Three Fiber
* React Drei

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Airplane-Simulator.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit the local address shown in the terminal (typically `http://localhost:5173`).
